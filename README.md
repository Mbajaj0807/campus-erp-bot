# Unofficial Campus ERP Automation Toolkit

> ⚠️ **Unofficial project** — built for learning and experimentation.  
> Not affiliated with or endorsed by any university or ERP provider.

## Overview

A Node.js + Express based automation toolkit that fetches and organizes student ERP data to reduce repetitive manual checks.

The project focuses on understanding real-world API systems, authentication flows, and backend data aggregation.

## What it Handles

- Attendance records  
- Timetable data  
- Leave / outpass status  
- Mess menu & daily schedules  

## Tech Stack

- Node.js  
- Express.js  
- REST APIs  
- Async / Promise-based architecture  
- JSON data processing  

## Purpose

This project was built to:
- Explore ERP-style backend systems
- Practice API-driven automation
- Design clean, modular backend services

## Disclaimer

This is an **unofficial, read-only automation project** created purely for educational purposes.  
No data is modified, bypassed, or misused.
